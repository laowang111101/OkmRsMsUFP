# 知识管理系统 java40

## 项目概述

知识管理系统是一个基于B/S架构的Web网站，采用MVC模式设计，实现了前后端分离。该系统主要用于知识的管理和知识库的建设。

## 技术栈

- 架构：B/S、MVC
- 开发环境：IDEA、JDK1.8、Maven、Mysql
- 技术语言：Java
- 数据库：Mysql
- 后端框架：SpringBoot、Mybatis
- 前端框架：Vue
